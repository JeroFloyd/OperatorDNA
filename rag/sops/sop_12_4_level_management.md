# SOP 12.4 — Vessel Level Management

**Revision:** 3.0
**Effective Date:** 2024-04-22
**Applicable To:** Process operators

## Purpose

Standard procedure for maintaining vessel level within safe operating limits.

## Normal Operating Range

- **Target Level:** 40-60%
- **Warning Range:** 15-25% or 75-85%
- **Critical Range:** <10% or >90%

## High Level Response (>80%)

1. **Verify level reading** — check redundant transmitter
2. **Increase outflow:**
   - Open discharge valve by 10-20%
   - If valve already >80%, increase pump speed by 200 RPM
3. **Monitor** — level should stabilize within 30 seconds
4. If level exceeds 90%, escalate to critical response

## Low Level Response (<20%)

1. **Verify level reading** — check redundant transmitter
2. **Reduce outflow:**
   - Close discharge valve by 10-20%
   - If valve already <20%, reduce pump speed by 200 RPM
3. **Monitor** — level should stabilize within 30 seconds
4. If level falls below 10%, execute pump protection: reduce pump to minimum RPM

## Normal Adjustment

For routine level adjustments:
- Change valve position by 5-10% increments
- Wait 15 seconds between adjustments
- Avoid large single adjustments (>20%)
